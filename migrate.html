Menu to Inventory Migration

Warning: This will move all documents from 'menu' to 'inventory' collection with stock = 10.
Inventory fields that will be created:
  • name - Item name
  • category - Category ID
  • price - Price in pesos
  • image - Image data
  • stock - Set to 10
  • status - available/unavailable
  • description - Item description
  • createdAt - Creation timestamp
  • updatedAt - Update timestamp
